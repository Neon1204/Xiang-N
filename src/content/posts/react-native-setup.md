# React Native 环境配置指南

React Native 是 Facebook 推出的跨平台移动应用开发框架。

## 环境要求

- Node.js >= 18
- JDK 17+
- Android Studio / Xcode

## 快速开始

```bash
npx react-native@latest init MyProject
cd MyProject
npm run android
```
