# NestJS 入门指南

NestJS 是一个用于构建高效、可扩展的 Node.js 服务器端应用程序的框架。

## 核心概念

- 模块 (Modules)
- 控制器 (Controllers)
- 提供者 (Providers)
- 依赖注入

## 快速开始

```bash
npm i -g @nestjs/cli
nest new project-name
```
