## 快速开始

### 本地开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

### 构建部署

```bash
# 构建静态文件
pnpm build

# 预览构建结果
pnpm preview
```

### 部署到 GitHub Pages

推送到 master 分支后，GitHub Actions 会自动部署到 GitHub Pages。
